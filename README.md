# Twilight.org

 Twilight.org is a full-stack **e-commerce platform backend** built using **Java 21** and **Spring Boot 3**.  
This project focuses on clean backend architecture, scalable APIs, and real-world system design concepts.

---

## Features

- User registration & authentication
- Product catalog management
- Order placement and order tracking
- Address management
- Payment workflow 
- RESTful API design
- Clean layered architecture

---

## Tech Stack

- **Java 21**
- **Spring Boot 3**
- **Spring Data JPA**
- **Spring Security**
- **Hibernate**
- **Maven**
- **MySQL / H2**
- **REST APIs**
